# Git Useful commands (CLI):

            1. git init
            2. git add f1.md
            3. git rm --cache f1.md   // ye staging area se working area mai jane k liye hai( aur .git  se woh file delete kar deta hai)
            3.1. git restore --staged meme.md   //ye command file ko ek step piche leke aatta hai

            #
            4. git commit -m "commit message"
            5. git commit --amend --no-edit  // commit hone k baad phir changes karna hai tab ye command 
                                                use karte hai
            6. git commit --amend -m "NEw Commit Message"   
            

            # rename of branch
            1. git branch -m branchname.// main aur master  (respectfully name change karna chahata hu)
            2. git branch -M master   (forcefully name change karna chahate hai )

             src/*.html :-  ye src k andar saree html file ignore karna hai
